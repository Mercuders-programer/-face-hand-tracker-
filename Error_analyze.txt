────────────────────────────────────────────────────────────────
  [SECTION 5] 원인 분석 — 왜 일반 confidence(0.5)에서 실패하는가
────────────────────────────────────────────────────────────────

  [원인 1] 얼굴 각도 — yaw = -68.6° (매우 측면)
    - FaceLandmarker 내부 BlazeFace는 정면(yaw ±30°) 학습 편향
    - yaw -68.6°는 거의 귀가 카메라를 향하는 각도
    - 이 각도에서 detection score가 매우 낮게 생성됨 (~0.03~0.04 추정)
    - conf=0.05 필터를 통과하려면 score > 0.05 이어야 함
    - 고해상도에서는 score가 임계값 미달, 소형 해상도에서는 경계값 돌파

  [원인 2] 해상도에 따른 score 차이
    - 고해상도(1080px): BlazeFace 입력 전 내부 리사이즈 → 상대적 패턴 다름
    - 소형 해상도(202x360): 얼굴 영역이 입력에서 차지하는 비율이 달라짐
    - 202x360에서만 conf=0.05를 간신히 넘는 score 생성됨
    - (마치 노이즈와 신호의 경계에 있는 상태)

  [원인 3] 두 단계 파이프라인 연쇄
    - Stage 1 (BlazeFace 검출): 후보 ROI 생성
    - Stage 2 (FaceMesh 회귀): 478pt 랜드마크 추출
    → Stage 1 score가 conf 미만이면 Stage 2 자체가 실행 안 됨
    → conf=0.5에서는 Stage 1이 완전히 실패

  [원인 4] 포즈와 달리 얼굴 감지기는 가림(occlusion)에 민감
    - PoseLandmarker는 측면 몸통 패턴도 학습됨 → 감지 성공
    - BlazeFace는 정면 얼굴 패턴에 특화 → 측면에서 score 급락

────────────────────────────────────────────────────────────────
  [SECTION 6] 해결 방안 비교
────────────────────────────────────────────────────────────────

  [방법 A] confidence 임계값 낮추기 (★ 즉시 적용 가능)
    구현: FaceLandmarker min_face_detection_confidence = 0.05
    장점: 코드 1줄 수정, 이 영상에서 감지 성공 확인됨
    단점: 오탐(false positive) 증가 가능
           고해상도에서는 여전히 0명 → 소형 해상도 같이 필요
    적용 파일: src/tracker.py (FaceLandmarker 생성 옵션)

  [방법 B] confidence + 소형 해상도 병행 (★★ 추천)
    구현: conf=0.05 + 감지용 프레임을 max_dim=360 으로 리사이즈
    장점: 이 영상에서 1명 감지 성공 (실험으로 확인)
    단점: 해상도가 낮아지므로 랜드마크 정밀도 다소 감소
    적용: tracker.py 감지용 리사이즈 로직 추가

  [방법 C] 포즈 head 랜드마크로 대체 (★★ 현실적)
    - 포즈 감지 성공 → nose(0), eye(1~6), ear(7~8) 재활용
    - 얼굴 bbox: nose ± (ear 간격 / 2) 추정
    - 모자이크 기능: 포즈 head bbox에 적용 가능
    - 단점: 478pt 정밀 랜드마크 없음, 얼굴 이미지 오버레이 불가

  [방법 D] RetinaFace 모델 교체
    - pip install retina-face
    - WIDER FACE 학습, yaw ±90° 감지 가능
    - 단점: MediaPipe 파이프라인과 별도 설치/유지

  [방법 E] 현재 상태 유지 (얼굴 감지 포기)

  ★ 최우선 권장: 방법 A+B (conf=0.05 + 360px 해상도 병행 적용)
    - 즉시 적용 가능, 추가 라이브러리 불필요
    - 이 영상에서 실험으로 효과 확인됨
