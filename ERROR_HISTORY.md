## [2026-04-12] retina-face 설치로 tensorflow 설치 → mediapipe segfault

### 오류 유형
의존성 충돌로 인한 환경 파괴

### 발생 상황
- insightface 설치 실패 → 대안으로 `pip install retina-face` 실행
- retina-face가 tensorflow 2.21.0 + keras를 자동 설치
- TF의 CUDA/GPU 관련 공유 라이브러리가 mediapipe의 EGL 처리와 충돌
- mediapipe FaceLandmarker.create_from_options() 에서 segfault (exit 139)

### 해결
`pip uninstall tensorflow keras -y --break-system-packages` → mediapipe 복구

### 재발 방지
- 새 패키지 설치 전 mediapipe 동작 여부 미리 확인
- TF/TF-관련 패키지는 mediapipe와 함께 설치 금지
- retina-face는 TF 의존이므로 사용 불가

## [2026-04-06] 배치파일 인코딩 오류 — make_exe_file.bat 실행 실패

### 오류 유형
Windows CMD 인코딩 불일치

### 증상
```
'e'은(는) 내부 또는 외부 명령... 아닙니다.
'windowed'은(는) 내부 또는 외부 명령... 아닙니다.
```

### 원인
1. 배치파일을 UTF-8로 저장 → Windows CMD는 CP949(한국어)로 파싱
2. `chcp 65001`은 출력 인코딩만 변경, 파일 파싱에는 무효
3. 한글 문자열이 깨지면서 명령어로 잘못 인식됨
4. PyInstaller 명령의 `^` 줄 이음이 깨진 인코딩으로 인해 실패

### 해결
- 배치파일 내 한글 전부 제거, ASCII 영문으로 교체
- `chcp 65001` 제거
- PyInstaller 명령을 한 줄로 합쳐 `^` 줄 이음 제거

### 재발 방지
- Windows .bat 파일은 한글 사용 금지, ASCII 영문만 사용
- PyInstaller 등 긴 명령은 한 줄로 작성
