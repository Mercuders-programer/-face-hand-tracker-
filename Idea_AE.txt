===================================================
PoseTracker — After Effects 활용 아이디어
===================================================

[ 내보내기 파일 구조 ]
ae_keyframes/
  person_0/
    face/   → 눈·코·입 12포인트 (x,y 픽셀)
    pose/   → 어깨·팔꿈치·손목·골반·무릎·발목 12관절
    hands/left|right/  → 손목·손가락 끝 6포인트 + 전체 21포인트

---------------------------------------------------
1. 얼굴 추적 (face/)
---------------------------------------------------
- 눈 위치에 이펙트/글로우 고정 (눈 빛나는 효과)
- 입(mouth_upper/lower)으로 립싱크 자동화 — 입 벌림 크기로 레이어 scale 제어
- 코 끝(nose_tip)에 텍스트·이모지·마스크 붙이기
- 눈·코·입 기반 얼굴 보정 레이어 추적 (NULL 오브젝트 연결)

---------------------------------------------------
2. 몸 추적 (pose/)
---------------------------------------------------
- 어깨(shoulder) → 상체 움직임에 옷·갑옷 레이어 연동
- 손목(wrist) → 검·총·소품이 손에 따라 움직이게
- 골반(hip) 중심 → 캐릭터 전체 이동 경로 추출
- 무릎·발목 → 걷기/점프 타이밍에 효과음·먼지 파티클 동기화
- 어깨↔어깨 거리로 카메라 줌 자동화 (Expression)

---------------------------------------------------
3. 손 추적 (hands/)
---------------------------------------------------
- 손가락 끝(index_tip) → 포인터/레이저 빔 효과
- 손목 위치 → 마법진·파티클 이펙트 고정
- 손 전체 21포인트 → 손 모션 블러·글로우 트레일

---------------------------------------------------
4. 다인원 (person_0, person_1 ...)
---------------------------------------------------
- 각 인물별 색상 코딩 레이어 분리
- 두 사람 손목 거리 → 인터랙션 이펙트 트리거 (Expression)
- 인물별 독립 모션 그래픽 오버레이

---------------------------------------------------
5. AE Import 방법
---------------------------------------------------
1) AE 메뉴: File > Scripts > Run Script File
2) scripts/import_to_ae.jsx 실행
   (또는 txt 파일을 직접 Edit > Paste Mocha mask 방식으로 수동 붙여넣기)
3) 생성된 NULL 오브젝트에 원하는 레이어를 Parent로 연결
