   - `elbow_perp = normalize(perp_up + perp_lo)` 계산 (대향 예외 처리 포함)
   - `n_blend = min(3, max(1, N // 2))` 혼합 구간 설정
   - Upper 루프: elbow 근방 `n_blend` 구간에서 `perp_up → elbow_perp` 선형 전환
   - Lower 루프: elbow 근방 `n_blend` 구간에서 `elbow_perp → perp_lo` 선형 전환

3. **`_build_arm_mesh` — `vert_blend` 코사인 스텝 계산**
   - cross-section별 blend 값 계산 후 L/R 양쪽에 복제 → `(2M,)` array
   - 코사인 스무딩으로 0→1 부드럽게 전환

4. **`_deform_mesh` — binary mask → 벡터화 blend**
   - `vert_blend` 매개변수 추가
   - M_up, M_lo 모두 계산 후 `b * dst_lo + (1-b) * dst_up` 보간

5. **`apply_puppet_warp` — `cache.vert_blend` 전달**
