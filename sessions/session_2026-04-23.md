## 작업: Puppet Pin 팔꿈치 끊김(gap) 수정 — Soft Feathered Mask

### 수정 파일
- `src/puppet_pin.py` — `build_segment_cache()` 만 수정

### 근본 원인
하드 이진 마스크(0/255)로 upper/lower 분리 후 독립 Affine 변환 →
팔꿈치가 꺾이면 M_up/M_lo 회전각 차이로 인해 경계에서 gap 발생.

### 해결
Soft feathered mask: 팔꿈치 주변 `feather_px` 폭에서 선형 그라데이션.
- `upper_w = clip((f - d) / (2f), 0, 1)`
- `lower_w = clip((f + d) / (2f), 0, 1)`
- 어디서나 `upper_w + lower_w = 1` → gap 원천 불가
- `feather_px` 미지정 시 세그먼트 길이의 18% (10~45px) 자동 계산

### 테스트 결과
- 문법 검사: OK
- import 테스트: OK
- 단위 테스트: ALL PASS
  - elbow 경계 alpha 합: 254 (기대 255±3)
  - 어깨쪽 upper 우세, 손목쪽 lower 우세 정상 확인
