### Puppet Pin MLS-Affine 재구현 ✅

**목표**: 2-세그먼트 독립 Affine → Moving Least Squares (MLS-Affine) 역방향 매핑으로 교체

**변경 파일**:
- `src/puppet_pin.py` — 전면 교체 (video_panel.py, camera_panel.py 변경 없음)

**핵심 변경**:
- `SegmentCache`: `img_upper`/`img_lower` 제거 → `img_orig` (BGRA 원본)으로 대체 (하위호환 필드 None으로 유지)
- `build_segment_cache()`: 마스킹 연산 제거, 원본 복사만 수행
- `_mls_affine_map()` 신규: 완전 벡터화 MLS-Affine (einsum 기반), scale_down=4로 저해상도 계산 후 업스케일
- `apply_puppet_warp()`: cv2.remap 기반 단일 패스로 교체
- `_pts_valid()` 신규: 목적지 핀 간격 검사
- `_seg_affine()` 제거 (더 이상 불필요)
- `_norm()`, `_rot90()` 유지

**MLS 알고리즘**:
- 역방향 매핑: 출력 픽셀 v → 소스 픽셀 u
- w_i = 1/max(||v-q_i||², 1), 가중 중심점 q*/p*, M_num/M_den(+λI) 계산
- 2×2 역행렬 해석적 공식 (numpy.linalg 미사용)
- λ=1.0 정규화로 팔 일직선 시 rank-1 M_den 안정화

**테스트 결과**:
- 문법/import → OK
- 단위 테스트 3케이스 (직선팔/꺾인팔/핀겹침) → ALL PASS
- `python3 app.py` 8초 실행 → 에러 없이 정상 시작

### Puppet Pin 외삽 버그 수정 ✅

**원인**: MLS-Affine의 수학적 특성상 팔 제어점(dst_pts)에서 멀어질수록
affine이 상수로 수렴(선형 외삽) → 소스 이미지가 크거나 배경이 불투명하면
캔버스 전체가 이미지로 채워짐

  최단 거리 계산 후 Gaussian 감쇠 마스크 생성
- σ = arm_length_canvas × 0.4 (최소 30px)
- alpha 채널에 마스크를 곱해 팔 영역 외부를 자동 투명 처리
