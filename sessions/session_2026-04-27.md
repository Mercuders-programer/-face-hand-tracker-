## 작업: Puppet Pin 세그먼트 폭 균일화

### 문제
각 세그먼트(upper/lower/hand)에 독립적인 Affine 변환 적용 시,
세그먼트별 로컬 스케일 차이(s_up ≠ s_lo)가 수직(폭) 방향에도 반영되어
elbow 접합부에서 폭이 불연속 → 세그먼트별 크기가 달라 보이는 문제.

### 해결
`src/puppet_pin.py` 수정:

1. **`_seg_affine` 함수** — `global_scale: Optional[float] = None` 파라미터 추가
   - `global_scale` 지정 시: 수직 방향 3번째 점 = `q0 + rot90(norm(d_v)) * global_scale * local_len`
   - `None`: 기존 동작 유지

2. **`_deform_mesh` 함수** — `_seg_affine` 호출 전 global_scale 계산
   - `global_scale = (vid_total_arc) / (img_total_arc)`
   - 모든 세그먼트에 동일한 global_scale 전달

### 검증
- AST OK
- shape 검증 OK (400×400×4)
- global_scale 계산 검증 OK (균형: 1.0, 불균형: 1.25)
- `timeout 6 python3 app.py` → 6초 정상 실행 후 종료

### 변경 파일
- `src/puppet_pin.py` (L323~340, L464~490)

## 작업: Puppet Pin 크기 슬라이더 적용 버그 수정

### 문제
Puppet Pin 모드에서 "크기 (%)" 슬라이더(30~300%)가 아무 효과 없는 문제.
- `apply_puppet_warp()` 호출 시 `size_pct` 미전달 (video_panel.py, camera_panel.py)
- `apply_puppet_warp()`, `_deform_mesh()`에 `size_pct` 파라미터 자체 없음

### 해결
1. `puppet_pin.py`: `apply_puppet_warp`에 `size_pct=100.0` 파라미터 추가, `_deform_mesh` 호출에 전달
2. `puppet_pin.py`: `_deform_mesh`에 `size_pct=100.0` 파라미터 추가, `global_scale *= size_pct / 100.0` 적용
3. `video_panel.py:250`: `apply_puppet_warp` 호출에 `size_pct=float(size_pct)` 추가
4. `camera_panel.py:252`: 동일

### 검증
- AST OK (3개 파일)
- width@50%: 50, width@100%: 100, width@200%: 200 (정확히 비례)
- `timeout 6 python3 app.py` → exit:124 (정상 실행 후 종료)

### 변경 파일
- `src/puppet_pin.py`
- `src/video_panel.py`
- `src/camera_panel.py`
