## 작업 내용

### EMA 평활화 — 얼굴 이미지 오버레이 떨림 수정

**문제:** 얼굴에 이미지를 붙이면 가만히 있어도 크기가 미세하게 커졌다 작아졌다를 반복하며 떨리는 현상.
**원인:** MediaPipe 랜드마크 좌표의 프레임 간 미세 노이즈가 스케일·위치·각도에 그대로 반영됨.

**수정 파일:** `src/camera_panel.py`

**변경 내용:**
1. `_ema_update()` 헬퍼 함수 추가 (모듈 레벨, 라인 69~77)
   - `smoothed = α × current + (1-α) × prev`, α = 0.15 기본값
   - `prev=None`이면 cold-start (current 그대로 저장)

2. `_apply_face_img_overlay()` 시그니처에 `ema_state: dict | None = None` 추가
   - 얼굴 없을 때: `ema_state` 내 모든 키를 None으로 리셋
   - Affine 자동 모드 내부: `face_h_px`, `eye_center(x,y)`, `angle` 4개 값 EMA 적용

3. `CameraPanel.__init__`에 EMA 상태 dict 추가:
   ```python
   self._face_img_ema = {'face_h': None, 'eye_cx': None, 'eye_cy': None, 'angle': None, 'alpha': 0.15}
   ```

4. `_toggle_face_image()`: 이미지 제거 시 EMA 상태 리셋

5. `_capture_loop()`: `_apply_face_img_overlay()` 호출 시 `ema_state=self._face_img_ema` 전달

**테스트 상태:** 문법 검사 통과. GUI 실행 테스트 필요 (영상 파일로 확인 요).

## 검증 방법
1. `python3 app.py` 실행 → 영상 파일 불러오기
2. 얼굴 이미지 오버레이 활성화
3. 얼굴 가만히 있을 때: 이미지 크기 안정 여부 확인
4. 얼굴 빠르게 움직일 때: 이미지가 따라가는지 확인
